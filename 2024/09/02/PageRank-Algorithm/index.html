随机过程

随机过程：随机变量的序列，如果我们固定某一个观察时刻t，则事物在时刻t出现的状态是随机的。

伯努利过程：
$$
X = { x_1, x_2, \ldots } = { x_n : n = 1, 2, \ldots ; x_n \in {1, 0} }
$$
从时间和状态空间的维度上，随机过程可以继续被划分：

  1. 离散时间：如果参数 n 取值于离散集，比如整数集Z 或非负整数集 N，则该过程是一个离散时间随机过程。
  2. 连续时间：如果参数n可以取任意实数值，则称其为连续时间随机过程。

对于每个随机变量 **xn**：

  1. 离散状态空间：如果每个随机变量 **xn**只能取有限个或可数无限个值，那么这个随机过程就称为离散状态空间随机过程。例如，伯努利过程中的随机变量只能取0或1两个值。
  2. 连续状态空间：如果每个随机变量 xn 可以取任意实数值或更高维度的值，那么就是连续状态空间随机过程。

马尔可夫链

马尔可夫链（Markov Chain）是一种特殊的随机过程，它在离散时间上进行，并且具有一个重要的特性——马尔可夫性质。

马尔可夫性质指的是，在给定当前状态的情况下，未来的状态仅依赖于当前状态，而不依赖于过去的历史状态。换句话说，一旦当前状态确定了，过去的状态就不再提供额外的信息来预测未来。

  • 状态空间S：可以是有限或无限。

  • 初始状态分布：马尔可夫链开始于某个初始状态的概率分布 π0。

  • 转移概率分布

时间齐次的马尔可夫链：转移概率分布与 t 无关。

转移概率矩阵

如状态空间 S 中仅有三种状态，则转移概率矩阵为 3*3 矩阵，元素定义为由一种状态转为另一种状态的概率。这样只要有了转移概率矩阵，我们就可以从一个过往的概率分布链式拓展下去。
