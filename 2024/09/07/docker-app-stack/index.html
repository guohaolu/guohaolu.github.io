Docker应用栈

Docker 应用栈（Application Stack）通常指的是在一个或多个 Docker 容器中运行的一组相互协作的应用和服务。这些应用和服务通常是通过 Docker Compose 或其他编排工具来定义和管理的，它们共同构成了一个完整的应用环境。
