hexo-博客搭建

发表于 2024-09-07 更新于 2024-09-18

安装流程

安装node.js

进入官网安装 node.js，并在安装过程中添加到 Path 路径。

安装cnpm
安装hexo

通过 cnpm 安装hexo。

初始化博客

创建 blog 目录，并进入此目录下。通过国内镜像完成 Hexo 博客的初始化。

更换主题
  1. 在 theme 目录下通过 git clone 新主题源代码。
  2. 打开 hexo 项目的 _config.yml 文件，修改 theme 参数为新主题名称（theme 目录下的文件名，如：next）。
  3. 在主题的_config.yml配置文件中修改站点的语言为zh-CN。

部署到GitHub站点

创建仓库

仓库名称为：guohaolu.github.io

创建git秘钥

通过 git 生成密钥，这里需要填写自己的邮箱。

复制输出的密钥信息。

添加 git 密钥，进入Settings--SSH--New SSH Key，在Key中粘贴输出信息，点击Add SSH key。

部署

进入 github.io 的项目，点击 Use SSH，复制 git 开头的地址。

进入 Hexo 的配置文件，修改 deploy 一栏。

配置 git 的配置

安装 hexo 博客部署的插件

进入 github.io 的项目，点击 Setting ——> Pages，进入 Github Pages一栏，源分支选择 main。

图床搭建

创建Github项目

创建 Github 图床项目：ImageHostingWebsite。

使用PicGo

下载 PicGo，在 Github 一栏中进行配置。Token 可以在Github - Settings - Personal access tokens 中生成，只选择repo选项就可以。

Typora设置

在 Typora - 偏好设置 中进行图片上传设置。
