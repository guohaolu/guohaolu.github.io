容器生命周期管理

（1）docker run

docker run [OPTIONS] IMAGE [COMMAND] [ARG...] 基于特定镜像创建容器，并根据命令来控制它

$ sudo docker run ubuntu:latest echo “Hello World”

根据tag为latest的ubuntu镜像，创建容器（容器id会随机）。运行echo “Hello World”命令后，容器会停止运行。

$ sudo docker run -i -t –name mytest ubuntu:latest /bin/bash

以交互模式、分配伪终端的形式创建容器（名称为mytest），并运行。
